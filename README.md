# W16 影片與聲音情緒辨識 App 專題網站

本網站依據 `W16_智慧生活科技應用_Cowork_影片聲音情緒辨識App_教學簡報.pdf` 製作，整理 AI 協作開發情緒辨識 App 的專題目標、開發流程、技術架構、成果展示與倫理提醒。

## 內容重點

- 影片與 YouTube 來源的情緒辨識 App 構想
- Gradio 與 Hugging Face Space 的雲端展示流程
- 影像、聲音與整合輸出的三線架構
- 前端瀏覽器端媒體訊號分析：可上傳影片或音訊，依聲音能量、語調變化、畫面亮度、色彩與動作幅度推估情緒分布
- AI 情緒辨識的限制、個資與倫理提醒

## 功能說明

目前公開網站可直接在瀏覽器執行分析，不會把影片或音訊上傳到伺服器。此版本適合作為課堂專題展示與原型驗證；若要達到正式 AI 模型等級，可再串接 Gradio / Hugging Face Space 的表情與聲音情緒模型。
